еңЁжұӮжңҖеӨ§е…¬зәҰж•°ж—¶пјҢеҹәдәҺдёҖдёӘеҹәжң¬зҡ„дәӢе®һ$(a, b) = (b, a - \lfloor \frac{a}{b}\rfloor b) = (b, a\ mod \ b)$пјҢзӣҙеҲ°$b = 0$пјҢжӯӨж—¶$(a, 0) = a$гҖӮиҝҷз§ҚжұӮжңҖеӨ§е…¬зәҰж•°зҡ„з®—жі•еҸ«еҒҡиҫ—иҪ¬зӣёйҷӨжі•пјҢеҸҲеҸ«еҒҡж¬§еҮ йҮҢеҫ—з®—жі•гҖӮ

иЈҙиңҖе®ҡзҗҶпјҡеҒҮи®ҫ$a, b$жҳҜдёҚе…Ёдёә0зҡ„ж•ҙж•°пјҢеҲҷеӯҳеңЁж•ҙж•°$x, y$дҪҝеҫ—$ax + by = gcd(a, b)$гҖӮ

з®ҖеҚ•иҜҒжҳҺпјҡ
$$
gcd(a, b) = gcd(b, a\ mod\ b) = gcd(b, a - kb) = gcd(r_1, r_2) = \dots = gcd(r_n, 0)
$$
е…¶дёӯ$r_1, r_2 \dots r_n$йғҪеҸҜд»ҘеҶҷжҲҗ$x_ia + y_ib$зҡ„еҪўејҸпјҢеӣ жӯӨеӯҳеңЁ$ax + by = gcd(a, b)$

жү©еұ•ж¬§еҮ йҮҢеҫ—з®—жі•еңЁжұӮи§ЈжңҖеӨ§е…¬зәҰж•°зҡ„еҗҢж—¶пјҢи®Ўз®—дәҶдёҖз»„йҖӮдәҺиЈҙиңҖе®ҡзҗҶзҡ„зі»ж•°гҖӮ
йҖ’жҺЁе…ізі»зҡ„жҺЁеҜјпјҢ$(a, b) = (b, a\ mod\ b) = (b, a - \lfloor \frac{a}{b} \rfloor b)$пјҢеӣ жӯӨеӯҳеңЁеӣӣдёӘж•ҙж•°$x, y, xвҖҷ, yвҖҷ$дҪҝеҫ—
$$
ax + by = bxвҖҷ + (a - \lfloor \frac{a}{b} \rfloor b)yвҖҷ \\
a(x - yвҖҷ) + b(y - (xвҖҷ - \lfloor \frac{a}{b} \rfloor)yвҖҷ) = 0 \\
x = yвҖҷ, y = xвҖҷ - \lfloor \frac{a}{b} \rfloor yвҖҷ
$$

дёҠиҝ°з®—жі•еҸҜд»Ҙж”№еҶҷдёәиҝӯд»Јзҡ„еҪўејҸпјҢйҖ’жҺЁе…¬ејҸ
$$
\left{ \begin{array}{rcl}
{\rm{x}} = yвҖҷ \\
{y = xвҖҷ - \lfloor \frac{a}{b} \rfloor yвҖҷ}
\end{array} \right.
$$
е°Ҷе…¶еҶҷдёәзҹ©йҳөзҡ„еҪўејҸ

$$
\left( \begin{array}{rcl}
x \\
y
\end{array} \right) = \left( \begin{array}{rcl}
0&1 \\
1&{ -\lfloor \frac{a}{b} \rfloor}
\end{array} \right)\left( \begin{array}{rcl}
{xвҖҷ} \\
{yвҖҷ}
\end{array} \right)
$$

  • еҲқе§Ӣзҹ©йҳөдёәеҚ•дҪҚзҹ©йҳө
  • $
    \left( \begin{array}{rcl}
    x&m\\
    y&n
    \end{array} \right) = \left( \begin{array}{rcl}
    x&m\\

    \end{array} \right)\left( \begin{array}{rcl}

д№ҳжі•йҖҶе…ғ

1. йҖҶе…ғзҡ„е®ҡд№ү

еҰӮжһңдёҖдёӘзәҝжҖ§еҗҢдҪҷж–№зЁӢ$ax \equiv 1 (mod\ b)$пјҢеҲҷз§°$x$дёә$a\ mod\ b$зҡ„йҖҶе…ғпјҢи®°дҪң$a^{-1}$гҖӮ

жЁЎиҝҗз®—и§„еҲҷдёҺеҹәжң¬еӣӣеҲҷиҝҗз®—зұ»дјјпјҢдҪҶжҳҜйҷӨжі•дҫӢеӨ–гҖӮеңЁACMз«һиөӣдёӯпјҢйҷӨжі•еҸ–жЁЎзҡ„иҝҗз®—иҰҒз”ЁеҲ°жұӮйҖҶе…ғпјҢд»ҺиҖҢеҫ—еҲ°$(a / b) \% c = (a * x) \% c$

иҜҒжҳҺпјҡе·ІзҹҘ$(b * x) \% c = 1$пјҢеҒҮи®ҫ$(a / b) \% c = y_1, (a * x) \% c = y_2$

еҲҷ

  • $a / b = k_1*c + y_1\quad(1)$
  • $a*x = k_2 * c + y_2\quad(2)$
  • $(1) -(2)$$\quad \rightarrow \quad a/b - a*x = (k_1 - k_2)*c + (y_1 - y_2)$
  • е·ҰеҸіеҗҢж—¶д№ҳд»Ҙb $\quad \rightarrow \quad$ $a - a * x * b = k * c * b + (y1 - y2) * b$
  • е·ҰеҸіжЁЎдёҠc $\quad \rightarrow \quad (y1 - y2) * b \% c = 0$
  • еӣ дёә$b \neq 0$, жүҖд»Ҙ$y_1 = y_2$

2. жұӮйҖҶе…ғ

иҙ№й©¬е°Ҹе®ҡзҗҶпјҢиӢҘpдёәзҙ ж•°пјҢдё”$gcd(a, p) = 1$пјҢеҲҷ$a^{p - 1} \equiv 1 (mod\ p)$

иҜҒжҳҺпјҡ

  • жһ„йҖ дёҖдёӘеәҸеҲ—$A={1, 2, 3, \dots p - 1}$пјҢ иҝҷдёӘеәҸеҲ—жңүеҰӮдёӢзҡ„жҖ§иҙЁ:
  • $\mathop{\Pi}\limits_{i = 1}^{n}A_i = \mathop{\Pi}\limits_{i = 1}^{n}(A_i \times a)$
  • еҜ№дәҺд»»ж„Ҹ$i, j \in n $пјҢ$(A_i * a )\% p \neq (A_j * a )\% p$пјҢеӣ дёә$(A_i - A_j) * a \% p = 0$пјҢеҲҷ$A_i - A_j = 0$
  • еӣ жӯӨ$(p - 1)! \equiv (p - 1)! * a^{p - 1}\ (mod\ p) \quad \rightarrow a^{p - 1} \equiv 1 \ mod(\ p)$

2.1 жү©еұ•ж¬§еҮ йҮҢеҫ—з®—жі•

жү©еұ•ж¬§еҮ йҮҢеҫ—з®—жі•

жұӮи§ЈеҗҢдҪҷж–№зЁӢ$ax \equiv 1 (mod\ b)$зҡ„жңҖе°Ҹж•ҙж•°и§Ј

еҒҮи®ҫйҮҮз”Ёжү©еұ•ж¬§еҮ йҮҢеҫ—з®—жі•жұӮеҫ—дёҖз»„$x_0, y_0$ж»Ўи¶і$ax_0 +by_0 = gcd(a, b)$пјҢеҲҷиҜҘж–№зЁӢзҡ„д»»ж„Ҹи§ЈеҸҜд»ҘиЎЁзӨәдёә$x = x_0 + bt, y = y_0-at$пјҢеҜ№дәҺд»»ж„ҸtеқҮжҲҗз«ӢпјҢжңҖе°Ҹж•ҙж•°и§ЈеҸҜд»ҘиЎЁзӨәдёә$x = ((x_0\ mod\ b) + b) mod\ b$

2.2еҝ«йҖҹе№Ӯз®—жі•

ж №жҚ®иҙ№й©¬е°Ҹе®ҡзҗҶпјҢ$1 \equiv a^{p -1} (mod\ p)$пјҢеӣ жӯӨ$x = a^{p - 2}$пјҢеҸҜд»ҘйҮҮз”Ёеҝ«йҖҹе№Ӯз®—жі•иҝӣиЎҢи®Ўз®—гҖӮ

2.3 зәҝжҖ§жұӮйҖҶе…ғ

жұӮеҮә$1, 2, \dots, n$дёӯжҜҸдёӘж•°е…ідәҺpзҡ„йҖҶе…ғгҖӮ

  • жҳҫз„¶$1{-1} \equiv 1 (mod\ p)$
  • д»Ө$k = \lfloor\frac{p}{i} \rfloor, j = p\ mod\ i$, еӣ жӯӨжңү$p = ki + j$пјҢеңЁжЁЎpзҡ„ж„Ҹд№үдёӢе°ұжңү$ki + j \equiv 0\ (mod\ p)$
  • е·ҰеҸіеҗҢд№ҳд»Ҙ$i^{-1}, j^{-1}$пјҢжңү$kj^{-1} + i^{-1}=0$
  • $i^{-1}=-kj^{-1} = (p - k)j^{-1}$

int [] shuffle()иҝ”еӣһж•°жҚ®йҡҸжңәжү“д№ұеҗҺзҡ„з»“жһңгҖӮ

жү“д№ұзҡ„жҰӮеҝө

  • йҡҸжңәжү“д№ұеҗҺзҡ„з»“жһңиғҪеӨҹиҰҶзӣ–жүҖжңүзҡ„жғ…еҶөпјҢжҜ”еҰӮж•°з»„е…ғзҙ зҡ„дёӘж•°дёәnпјҢйӮЈд№Ҳжү“д№ұеҗҺзҡ„з»“жһңеә”иҜҘиҰҶзӣ–$n!$з§ҚжүҖжңүеҸҜиғҪзҡ„з»“жһң

  • жүҖжңүз»“жһңзӯүжҰӮзҺҮеҮәзҺ°

ж–№жі•дёҖ жҡҙеҠӣ

жҡҙеҠӣзҡ„ж–№жі•е°ұжҳҜжЁЎжӢҹпјҢеҒҮи®ҫж•°з»„дёӯзҡ„жҜҸдёӘе…ғзҙ дёәдёҖдёӘе°ҸзҗғпјҢжҲ‘д»¬е°Ҷе…¶ж”ҫе…ҘиўӢеӯҗдёӯпјҢиҝӣиЎҢдёҚж”ҫеӣһзҡ„жҠҪеҸ–пјҢзӣҙеҲ°иўӢеӯҗдёӯжІЎжңүе°ҸзҗғпјҢжҲ‘д»¬е°ҶжҠҪеҸ–еҮәжқҘзҡ„е°ҸзҗғжҢүйЎәеәҸжҺ’еҲ—еҫ—еҲ°дёҖдёӘжҺ’еҲ—з»„еҗҲпјҢиҝҷдёӘжҺ’еҲ—з»„еҗҲеҮәзҺ°зҡ„жҰӮзҺҮдёә$\frac{1}{n!}$пјҢеӣ жӯӨиҝҷдёӘ ж•°з»„зҡ„жҜҸдёӘжҺ’еҲ—з»„еҗҲйғҪжҳҜзӯүжҰӮзҡ„пјҢз®—жі•ж—¶й—ҙеӨҚжқӮеәҰ$O(n2)$пјҢvector<int>зҡ„erase()еҲ йҷӨж“ҚдҪңжҳҜO(n)зҡ„гҖӮ

еҜ№дәҺжҹҗдёҖдёӘжҺ’еҲ—з»„еҗҲе…¶еҮәзҺ°жҰӮзҺҮдёә
$$
\mathop{\Pi}\limits_{k = 0}^{n - 1}\frac{1}{n - k} = \frac{1}{n!}
$$

ж–№жі•дәҢ Fisher-Yatesз®—жі•

Fisher-Yates жҙ—зүҢз®—жі•и·ҹжҡҙеҠӣз®—жі•еҫҲеғҸгҖӮеңЁжҜҸж¬Ўиҝӯд»ЈдёӯпјҢз”ҹжҲҗдёҖдёӘеҪ“еүҚж•°з»„жң«е°ҫжҢҮй’ҲдёӢж Үзҡ„йҡҸжңәж•ҙж•°гҖӮжҺҘдёӢжқҘпјҢе°ҶеҪ“еүҚе…ғзҙ е’ҢйҡҸжңәйҖүеҮәзҡ„дёӢж ҮжүҖжҢҮзҡ„е…ғзҙ дә’зӣёдәӨжҚўпјҢиҝҷжӯҘжЁЎжӢҹдәҶжҜҸж¬Ўд»ҺиўӢеӯҗйҮҢйқўж‘ёдёҖдёӘе…ғзҙ зҡ„иҝҮзЁӢгҖӮ

еҮёдјҳеҢ–

еҸ‘иЎЁдәҺ 2021-01-12 еҲҶзұ»дәҺ еҮёдјҳеҢ–

еҮёдјҳеҢ–

leetcode 1515 жңҚеҠЎдёӯеҝғзҡ„жңҖдҪідҪҚзҪ®

дёҖе®¶еҝ«йҖ’е…¬еҸёеёҢжңӣеңЁж–°еҹҺеёӮе»әз«Ӣж–°зҡ„жңҚеҠЎдёӯеҝғгҖӮе…¬еҸёз»ҹи®ЎдәҶиҜҘеҹҺеёӮжүҖжңүе®ўжҲ·еңЁдәҢз»ҙең°еӣҫдёҠзҡ„еқҗж ҮпјҢе№¶еёҢжңӣиғҪеӨҹд»ҘжӯӨдёәдҫқжҚ®дёәж–°зҡ„жңҚеҠЎдёӯеҝғйҖүеқҖпјҡдҪҝжңҚеҠЎдёӯеҝғ еҲ°жүҖжңүе®ўжҲ·зҡ„ж¬§еҮ йҮҢеҫ—и·қзҰ»зҡ„жҖ»е’ҢжңҖе°Ҹ гҖӮз»ҷдҪ дёҖдёӘж•°з»„ positions пјҢе…¶дёӯ positions[i] = [xi, yi] иЎЁзӨәз¬¬ i дёӘе®ўжҲ·еңЁдәҢз»ҙең°еӣҫдёҠзҡ„дҪҚзҪ®пјҢиҝ”еӣһеҲ°жүҖжңүе®ўжҲ·зҡ„ ж¬§еҮ йҮҢеҫ—и·қзҰ»зҡ„жңҖе°ҸжҖ»е’Ң

иҜҘй—®йўҳжң¬иҙЁдёҠдјҳеҢ–зҡ„еҮҪж•°еҰӮдёӢпјҡ
$$
f(x_c, y_c) = \sum\limits_{i = 0}^{n - 1}\sqrt{(x_c - x_i)^2 + (y_c - y_i)^2}
$$
е…¶дёӯ$(x_c, y_c)$дёәжңҚеҠЎеҷЁдёӯеҝғзҡ„дҪҚзҪ®пјҢе…¶еҸҜд»ҘдҪҝ$f(x_c, y_c)$еҸ–еҫ—жңҖе°ҸеҖјгҖӮиҜҘеҮҪж•°жҳҜеҮёеҮҪж•°пјҢеҮёдјҳеҢ–й—®йўҳзҡ„еұҖйғЁжңҖдјҳи§Је°ұжҳҜе…ЁеұҖжңҖдјҳи§ЈгҖӮ

1.еҮёйӣҶ

$C$жҳҜеҮёйӣҶпјҢеҰӮжһңеҜ№дәҺд»»ж„Ҹзҡ„$x,y\in C$е’Ңд»»ж„Ҹзҡ„$\theta \in \mathbb{R}$дё”ж»Ўи¶і$0 \le \theta \le 1$ж—¶пјҢ$\theta x + (1 - \theta) y$жҒ’жҲҗз«ӢгҖӮ

2.еҮёеҮҪж•°

2.1 еҮёеҮҪж•°е®ҡд№ү

е®ҡд№үеңЁ$\mathbb{R}^n \rightarrow \mathbb{R}$дёҠзҡ„еҮҪж•°$f$жҳҜеҮёеҮҪж•°пјҢ еҪ“дё”д»…еҪ“е®ғзҡ„е®ҡд№үеҹҹ$\mathbb{D}(f)$жҳҜдёҖдёӘеҮёйӣҶпјҢдё”еҜ№дәҺд»»ж„Ҹзҡ„$x, y \in \mathbb{D}$е’Ң$ 0 \le \theta \le 1$пјҢ $f(\theta x + (1 - \theta)y) \le \theta f(x) + (1 - \theta)f(y)$жҒ’жҲҗз«ӢгҖӮ

2.2 еҮёеҮҪж•°зҡ„дёҖйҳ¶е……иҰҒжқЎд»¶

еҒҮи®ҫе®ҡд№үеңЁ$\mathbb{R}^n \rightarrow \mathbb{R}$дёҠзҡ„еҮҪж•°$f$еҸҜеҫ®пјҢ еҚіеҜ№дәҺжүҖжңүзҡ„$x \in \mathbb{D}(f)$пјҢжўҜеәҰ$\nabla f(x)$еқҮеӯҳеңЁпјҢеҲҷеҮҪж•°$f$жҳҜеҮёеҮҪж•°еҪ“дё”д»…еҪ“еҮҪж•°е®ҡд№үеҹҹ$\mathbb{D}(f)$жҳҜдёҖдёӘеҮёйӣҶеҗҲпјҢдё”еҜ№дәҺжүҖжңү$x, y \in \mathbb{D}(f)$еқҮж»Ўи¶іпјҡ
$$
f(y) \ge f(x) + \nabla f(x)^T(y - x)
$$

2.3 еҮёеҮҪж•°зҡ„дәҢйҳ¶е……иҰҒжқЎд»¶

и®°еҮёеҮҪж•°$f$зҡ„дёҖйҳ¶еҜјж•°е’ҢдәҢйҳ¶еҜјж•°еҲҶеҲ«дёә$g$е’Ң$H$
$$
\nabla f = \begin{bmatrix} \frac{\partial f}{\partial x_1} \\ \frac{\partial f}{\partial x_2}\\ \vdots \\ \frac{\partial f}{\partial x_n} \end{bmatrix} \quad
H = \nabla^2f = \begin{bmatrix} \frac{\partial^2f}{\partial x_1^2} & \frac{\partial^2f}{\partial x_1 \partial x_2} & \cdots & \frac{\partial^2f}{\partial x_1 \partial x_n} \\ \frac{\partial^2f}{\partial x_2 \partial x_1} & \frac{\partial^2f}{\partial x_2^2} & \cdots & \frac{\partial^2f}{\partial x_2 \partial x_n} \\ \vdots & \vdots & \ddots & \vdots \\ \frac{\partial^2f}{\partial x_n \partial x_1} & \frac{\partial^2df}{\partial x_n \partial x_2} & \cdots & \frac{\partial^2f}{\partial x_n^2} \end{bmatrix}
$$
$f$еңЁ$x_k$еӨ„зҡ„дәҢйҳ¶жі°еӢ’еұ•ејҖдёә
$$
f(x) = f(x_k) + \nabla f^T (x - x_k) + \frac{1}{2!}(x - x_k)H(x_k)(x - x_k) + o^n
$$
еҒҮи®ҫе®ҡд№үеңЁ$\mathbb{R}^n \rightarrow \mathbb{R}$дёҠзҡ„еҮҪж•°$f$дәҢйҳ¶еҸҜеҫ®пјҢ еҚіеҜ№дәҺжүҖжңүзҡ„$x \in \mathbb{D}(f)$пјҢжө·жЈ®зҹ©йҳө$\nabla^2 f(x)$еқҮеӯҳеңЁпјҢеҲҷеҮҪж•°$f$жҳҜеҮёеҮҪж•°еҪ“дё”д»…еҪ“еҮҪж•°е®ҡд№үеҹҹ$\mathbb{D}(f)$жҳҜдёҖдёӘеҮёйӣҶеҗҲпјҢдё”еҜ№дәҺжүҖжңү$x, y \in \mathbb{D}(f)$еқҮж»Ўи¶і$H$дёәеҚҠжӯЈе®ҡзҹ©йҳөгҖӮ

3.еҮёдјҳеҢ–

еҸҜд»ҘиҜҒжҳҺжң¬йўҳдјҳеҢ–зҡ„еҮҪж•°$f$дёә$\mathbb{R}^2$зҡ„дёҠеҮёеҮҪж•°гҖӮ

  • жўҜеәҰдёӢйҷҚ

  • жЁЎжӢҹйҖҖзҒ«

  • зҲ¬еұұз®—жі•

  • дёүеҲҶжҗңзҙў

3.1 жўҜеәҰдёӢйҷҚ

жўҜеәҰдёӢйҷҚз®—жі•пјҢжўҜеәҰдёӢйҷҚжҳҜжңәеҷЁеӯҰд№ дёӯеёёз”Ёзҡ„дёҖз§ҚжұӮеұҖйғЁжңҖе°ҸеҖјзҡ„з®—жі•пјҢеҜ№дәҺз»ҷе®ҡзӮ№$(x, y)$пјҢе®ғзҡ„жўҜеәҰж–№еҗ‘жҳҜеҮҪж•°еҖјдёҠеҚҮжңҖеҝ«зҡ„ж–№еҗ‘пјҢеӣ жӯӨиҙҹжўҜеәҰдёәеҮҪж•°еҖјдёӢйҷҚзҡ„жңҖеҝ«зҡ„ж–№еҗ‘пјҢ$-\nabla f = (-\frac{\partial f}{\partial x}, -\frac{\partial f}{\partial y})$пјҢжҲ‘д»¬д»ҺдёҖдёӘеҲқе§ӢзӮ№$(x_{start}, y_{start})$ејҖе§Ӣиҝӯд»ЈпјҢ жҜҸж¬Ўд»Ө
$$
x \prime = x - \alpha \cdot \frac{\partial f}{\partial x} \\
y \prime = y - \alpha \cdot \frac{\partial f}{\partial y}
$$

  1. еҲқе§ӢзӮ№$(x_{start}, y_{start})=(\frac{\sum\limits_{i =0}^{n-1}x_i}{n},\frac{\sum\limits_{i =0}^{n-1}y_i}{n})$

  2. еӯҰд№ зҺҮ$\alpha = 1$

  3. еӯҰд№ зҺҮиЎ°еҮҸ$\eta = 0.001$

  4. еҪ“$(x \prime, y\prime)$дёҺ$(x, y)$зҡ„и·қзҰ»е°ҸдәҺ$10^{-7}$ж—¶з»“жқҹиҝӯд»ЈгҖӮ

ж•ҙдҪ“еҶҷдёӢжқҘпјҢе°қиҜ•дәҶеҫҲеӨҡж¬ЎпјҢеҲқе§ӢеҖјдёҺеӯҰд№ зҺҮдёҺеӯҰд№ зҺҮиЎ°еҮҸзҡ„и®ҫзҪ®йғҪдјҡеҫҲеӨ§зЁӢеәҰдёҠеҪұе“ҚжҳҜеҗҰиғҪеӨҹж”¶ж•ӣеҲ°жңҖдјҳеҖјгҖӮиҜҘз®—жі•еҜ№еҲқе§ӢеҢ–еҸӮж•°и®ҫзҪ®зҡ„дҫқиө–иҫғеӨ§гҖӮ

3.2 зҲ¬еұұз®—жі•

еҰӮжһңз»ҷе®ҡзҡ„еҮёеҮҪж•°еҫҲйҡҫжұӮеҜјпјҢжҲ‘д»¬жіЁж„ҸеҲ°иҙҹжўҜеәҰж–№еҗ‘пјҢ$\nabla f = -\frac{\partial f}{\partial x}(1, 0) -\frac{\partial f}{\partial y}(0, 1)$пјҢејҖе§ӢжҲ‘д»¬йҖүжӢ©дёҖдёӘжӯҘй•ҝпјҢиЎЁзӨәжҜҸж¬Ўз§»еҠЁзҡ„и·қзҰ»гҖӮеҰӮжһңжҲ‘д»¬еҪ“еүҚеңЁдҪҚзҪ®$(x, y)$пјҢжҲ‘д»¬дёҖж¬ЎжһҡдёҫеӣӣдёӘж–№еҗ‘пјҢеҲӨж–ӯе…¶еҮҪж•°еҖјжҳҜеҗҰжӣҙе°ҸпјҢеҰӮжһңжӣҙе°ҸеҲҷиҝӣиЎҢз§»еҠЁпјҢеҗҰеҲҷиҜҙжҳҺжҲ‘д»¬зҡ„жӯҘй•ҝиҝҮеӨ§пјҢзӣҙжҺҘи·іиҝҮдәҶжңҖдјҳеҖјзӮ№пјҢи°ғж•ҙжӯҘй•ҝдёәеҺҹжқҘзҡ„дёҖеҚҠпјҢзӣҙеҲ°жӯҘй•ҝе°ҸдәҺз»ҷе®ҡзҡ„йҳҲеҖјгҖӮ

е№¶жҹҘйӣҶ

еҸ‘иЎЁдәҺ 2021-01-06 еҲҶзұ»дәҺ Data Structure

е№¶жҹҘйӣҶ

е№¶жҹҘйӣҶеҸҜд»ҘиҜҙеңЁз®—жі•дёӯеә”з”ЁеҫҲе№ҝжіӣпјҢжңҖдё»иҰҒз”ЁжқҘеҲӨж–ӯеӣҫзҡ„иҝһйҖҡжҖ§зҡ„й—®йўҳгҖӮд№ӢеүҚеҒҡзҡ„йўҳеҫҲе°‘еҢ…еҗ«жҢүз§©еҗҲе№¶пјҢжңҖеӨҡе°ұжҳҜжҢүз…§дёӨдёӘиҝһйҖҡеҲҶйҮҸзҡ„еӨ§е°ҸиҝӣиЎҢеҗҲе№¶гҖӮд»ҠеӨ©з¬¬дёҖж¬Ўзў°еҲ°дәҶжҢүз…§з§©иҝӣиЎҢеҗҲе№¶зҡ„е№¶жҹҘйӣҶй—®йўҳпјҢleetcode 399 йҷӨжі•жұӮеҖјпјҢжҸҸиҝ°еҰӮдёӢпјҡ

з»ҷдҪ дёҖдёӘеҸҳйҮҸеҜ№ж•°з»„ equations е’ҢдёҖдёӘе®һж•°еҖјж•°з»„ values дҪңдёәе·ІзҹҘжқЎд»¶пјҢе…¶дёӯ equations[i] = [Ai, Bi] е’Ң values[i] е…ұеҗҢиЎЁзӨәзӯүејҸ Ai / Bi = values[i] гҖӮжҜҸдёӘ Ai жҲ– Bi жҳҜдёҖдёӘиЎЁзӨәеҚ•дёӘеҸҳйҮҸзҡ„еӯ—з¬ҰдёІгҖӮ

еҸҰжңүдёҖдәӣд»Ҙж•°з»„ queries иЎЁзӨәзҡ„й—®йўҳпјҢе…¶дёӯ queries[j] = [Cj, Dj] иЎЁзӨәз¬¬ j дёӘй—®йўҳпјҢиҜ·дҪ ж №жҚ®е·ІзҹҘжқЎд»¶жүҫеҮә Cj / Dj = ? зҡ„з»“жһңдҪңдёәзӯ”жЎҲгҖӮ

иҫ“е…Ҙпјҡequations = [[вҖңaвҖқ,вҖқbвҖқ],[вҖңbвҖқ,вҖқcвҖқ]], values = [2.0,3.0], queries = [[вҖңaвҖқ,вҖқcвҖқ],[вҖңbвҖқ,вҖқaвҖқ],[вҖңaвҖқ,вҖқeвҖқ],[вҖңaвҖқ,вҖқaвҖқ],[вҖңxвҖқ,вҖқxвҖқ]]
иҫ“еҮәпјҡ[6.00000,0.50000,-1.00000,1.00000,-1.00000]

1.ж·ұеәҰдјҳе…Ҳжҗңзҙў

еӣ дёәж•°жҚ®и§„жЁЎдёҚеӨ§пјҢa / b = 2пјҢеҸҜд»ҘиЎЁзӨәдёәaеҲ°bжңүдёҖжқЎжқғеҖјдёә2зҡ„иҫ№пјҢbеҲ°aжңүдёҖжқЎжқғеҖјдёә0.5зҡ„иҫ№пјҢеӣ жӯӨеҜ№дәҺжҜҸдёҖдёӘжҹҘиҜўпјҢеҸҜд»ҘйҮҮз”Ёж·ұеәҰдјҳе…Ҳжҗңзҙўзҡ„ж–№ејҸиҝӣиЎҢжҹҘжүҫпјҢи®°еҪ•и·Ҝеҫ„дёҺеҪ“еүҚзҡ„д№ҳз§ҜпјҢж—¶й—ҙеӨҚжқӮеәҰдёәO(n)гҖӮ

2.е№¶жҹҘйӣҶж–№жі•

  • a / b = 2.0 иҜҙжҳҺa дёҺ bеңЁдёҖдёӘйӣҶеҗҲдёӯпјҢa = 2b

  • еёҰи·Ҝеҫ„еҺӢзј©зҡ„жҹҘиҜў
    $$
    w[x] = \frac{v[x]}{v[father} \
    w[x] = \frac{v[x]}{v[fa[x]]} * \frac{v[fa[x]]}{v[father]} \
    w[x] = w[x] * w[fa[x]]
    $$

  • жҢүз§©еҗҲе№¶

    еҗҲе№¶дёӨдёӘиҠӮзӮ№зҡ„зҲ¶дәІпјҢиҰҒжӣҙж–°дёӨдёӘзҲ¶дәІд№Ӣй—ҙзҡ„жқғеҖјгҖӮ
    $$
    w[fx] = \frac{v[fx]}{v[fy]} \
    w[fx] = \frac{v[x]}{w[x]} / \frac{v[y]}{w[y]} \
    $$

жҙӣи°·p1122 жңҖеӨ§еӯҗж ‘е’Ң

йўҳзӣ®дёӯз»ҷеҮәдәҶNдёӘзӮ№зҡ„жқғеҖјпјҢдёҺN-1жқЎиҫ№зҡ„иҝһжҺҘе…ізі»пјҢе№¶жңӘз»ҷеҮәж №гҖӮ

$1 \le N \le 16000$

ејҖе§ӢжҖ»жҳҜжғійңҖдёҚйңҖиҰҒжҚўж №пјҢиҜҘйўҳжң¬иҙЁдёҠжҳҜжұӮж ‘дёҠзӮ№ жқғеҖје’ҢжңҖеӨ§зҡ„дёҖдёӘиҝһйҖҡеҲҶйҮҸпјҢеӣ жӯӨйҖүжӢ©е“ӘдёӘзӮ№дёәж №еҜ№з»“жһңжІЎжңүеҪұе“ҚпјҢд»»дёҖиҝһйҖҡеҲҶйҮҸеңЁжҹҗдёҖж—¶еҲ»жҖ»жҳҜеҸҜд»Ҙи§ҶдёәдёҖйў—д»ҘжҹҗдёӘзӮ№дёәж №зҡ„ж•°гҖӮ
$$
f[u] += max(f[v], 0) \
ans = max(ans, f[u])
$$

жҙӣи°· p1613 и·‘и·Ҝ

йўҳзӣ®еӨ§ж„ҸпјҢеӣҫз»“жһ„пјҢеҰӮжһңiеҲ°jзҡ„и·қзҰ»дёә$2^k$еҚғзұіпјҢйӮЈд№ҲеҸӘйңҖиҰҒ1sж—¶й—ҙгҖӮ

$n \le 50, m \le 10000$

еӣ дёәж¶үеҸҠеҲ°$2^k$пјҢеӣ жӯӨдёәеҖҚеўһз®—жі•пјҢе…ҲиҝӣиЎҢйў„еӨ„зҗҶпјҢзҠ¶жҖҒиЎЁзӨәf[i][j][k]иЎЁзӨәiеҲ°jжңүдёҖжқЎи·қзҰ»дёә$2^k$зҡ„иҫ№гҖӮ
$$
f[i][j][k] = f[i][l][k - 1] && f[l][j][k - 1] \
1 \le l \le n
$$

ж ‘дёҺеӣҫдёҠзҡ„еҠЁжҖҒи§„еҲ’II

еҸ‘иЎЁдәҺ 2020-12-28 еҲҶзұ»дәҺ Data Structure

жҙӣи°·P2014 йҖүиҜҫ

еңЁеӨ§еӯҰйҮҢжҜҸдёӘеӯҰз”ҹпјҢдёәдәҶиҫҫеҲ°дёҖе®ҡзҡ„еӯҰеҲҶпјҢеҝ…йЎ»д»ҺеҫҲеӨҡиҜҫзЁӢйҮҢйҖүжӢ©дёҖдәӣиҜҫзЁӢжқҘеӯҰд№ пјҢеңЁиҜҫзЁӢйҮҢжңүдәӣиҜҫзЁӢеҝ…йЎ»еңЁжҹҗдәӣиҜҫзЁӢд№ӢеүҚеӯҰд№ пјҢеҰӮй«ҳзӯүж•°еӯҰжҖ»жҳҜеңЁе…¶е®ғиҜҫзЁӢд№ӢеүҚеӯҰд№ гҖӮзҺ°еңЁжңү NNN й—ЁеҠҹиҜҫпјҢжҜҸй—ЁиҜҫжңүдёӘеӯҰеҲҶпјҢжҜҸй—ЁиҜҫжңүдёҖй—ЁжҲ–жІЎжңүзӣҙжҺҘе…Ҳдҝ®иҜҫпјҲиӢҘиҜҫзЁӢ a жҳҜиҜҫзЁӢ b зҡ„е…Ҳдҝ®иҜҫеҚіеҸӘжңүеӯҰе®ҢдәҶиҜҫзЁӢ aпјҢжүҚиғҪеӯҰд№ иҜҫзЁӢ bпјүгҖӮдёҖдёӘеӯҰз”ҹиҰҒд»ҺиҝҷдәӣиҜҫзЁӢйҮҢйҖүжӢ© MMM й—ЁиҜҫзЁӢеӯҰд№ пјҢй—®д»–иғҪиҺ·еҫ—зҡ„жңҖеӨ§еӯҰеҲҶжҳҜеӨҡе°‘пјҹ

жҜҸй—ЁиҜҫеҸӘжңүдёҖй—Ёе…Ҳдҝ®иҜҫпјҢ еӣ жӯӨиҜҘеӣҫз»“жһ„дёәжЈ®жһ—пјҢеҒҮи®ҫжІЎжңүйҖүдҝ®иҜҫзҡ„зҲ¶иҠӮзӮ№йғҪжҢҮеҗ‘йӣ¶пјҢеӣ жӯӨеҰӮжһңиҰҒйҖүжӢ©еӯҗиҠӮзӮ№пјҢеҲҷеҝ…йЎ»иҰҒйҖүжӢ©зҲ¶иҠӮзӮ№пјҢеӣ дёәзҲ¶иҠӮзӮ№дёәйҖүдҝ®иҜҫзҡ„иҰҒжұӮгҖӮ

иҪ¬жҚўдёәж ‘дёҠиғҢеҢ…й—®йўҳ
$$
dp[i][j] = max(dp[i][j], dp[i][j - k] + dp[x][k]) \quad xдёәiзҡ„еӯҗиҠӮзӮ№гҖӮ
$$

жҙӣи°·p 2015 дәҢеҸүиӢ№жһңж ‘

дәҢеҸүж ‘е…ұжңүNдёӘз»“зӮ№пјҢзј–еҸ·дёә1-NпјҢж ‘ж №зј–еҸ·дёҖе®ҡжҳҜ1гҖӮжҜҸйў—ж ‘жһқдёҠжңүдёҖе®ҡж•°йҮҸиӢ№жһңпјҢз»ҷе®ҡйңҖиҰҒдҝқз•ҷзҡ„ж ‘жһқж•°йҮҸпјҢжұӮеҮәжңҖеӨҡиғҪз•ҷдҪҸеӨҡе°‘иӢ№жһңгҖӮ

$1\le N, Q \le 100$

йҖҡеёёе®ҡд№үеӣҫзҡ„ж–№ејҸдёәпјҡe[tot]и®°еҪ•дәҶжҹҗжқЎиҫ№зҡ„иҝһжҺҘдҝЎжҒҜпјҢhead[u]и®°еҪ•дәҶиҫ№зҡ„еәҸеҸ·гҖӮ

зҠ¶жҖҒе®ҡд№үпјҡ

зҠ¶жҖҒиҪ¬з§»ж–№зЁӢпјҢnйў—ж ‘зҡ„01иғҢеҢ…й—®йўҳгҖӮ
$$
f[u][i] = max(f[u][i], f[u][i - j - i] + f[v][j] + e[u->v].w)
$$
дҝқз•ҷдёҖжқЎиҫ№еҝ…йЎ»дҝқз•ҷд»Һж №иҠӮзӮ№еҲ°иҝҷжқЎиҫ№и·Ҝеҫ„дёҠзҡ„жүҖжңүиҫ№пјҢиӢҘжһңиҰҒдҝқз•ҷеӯҗж ‘vдёҠзҡ„иҫ№пјҢйӮЈд№Ҳеҝ…йЎ»дҝқз•ҷuеҲ°vзҡ„иҝҷжқЎиҫ№гҖӮ
